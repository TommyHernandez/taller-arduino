Quiénes somos

Somos la asociación Alpha-Bug, nos fundamos en Enero de 2012 con el fin de hacer llegar el Software libre en particular y la informática en general a todo el mundo.

Codigo Abierto

Código abierto es la expresión con la que se conoce al software o hardware distribuido y desarrollado libremente. Se focaliza más en los beneficios prácticos que en cuestiones éticas o de libertad.

Open Hardware

Son aquellos dispositivos de hardware cuyas especificaciones y diagramas esquemáticos son de acceso público, ya sea bajo algún tipo de pago, o de forma gratuita

¿Que es Arduino?

Arduino es una placa con un microcontrolador y un entorno de desarrollo cuyo objetivo es facilitar el desarrolo de proyectos electónicos.

Arduino Pro

Microcontrolador ATmega168 o ATmega328
Voltaje operativo 3.3V o 5V
Input voltage 3.35 -12 V o 5 - 12V
E/S Digitales 14 (de las cuales 6 son salidas PWM)
Entradas analógicas 6
Memoria Flash 16 KB (ATmega168) o 32KB (ATmega328) 2KB para el bootloader
SRAM 1 KB (ATmega168) o 2 KB (ATmega328)
EEPROM 512 bytes (ATmega168) o 1 KB (ATmega328)
Velocidad del Reloj 8MHz en 3.3V o 16MHz en 5V

Arduino Mega

Microcontrolador ATmega2560
Voltaje operativo 5V
Input voltage 7-12v recomendado pero admite entre 6v a 20v
E/S Digitales 54 (de las cuales 15 son salidas PWM)
Entradas analógicas 16
Memoria Flash 256KB 8KB para el bootloader
SRAM 8KB
EEPROM 4KB
Velocidad del Reloj 16MHz

Arduino Zero

Microcontrolador ATSAMD21G18, 32-Bit ARM Cortex M0+
Voltaje operativo 3.3V
E/S Digitales 20
Entradas analógicas 6, 12-bit ADC
Memoria Flash 256KB
SRAM 32KB
EEPROM No posee
Velocidad del Reloj 48MHz

Arduino Due

Microcontrolador AT91SAM3X8E
Voltaje operativo 3.3V
Voltage de entrada 7-12v recomendado pero admite entre 6v a 16v
E/S Digitales 54 (de las cuales 12 son salidas PWM)
Entradas analógicas 12
Memoria Flash 512KB
SRAM 96KB repartidos en dos bacos uno de 64KB y otro de 32KB
EEPROM --
Velocidad del Reloj 84MHz

Arduino Yún

Microcontrolador ATSAMD21G18, 32-Bit ARM Cortex M0+
Voltaje operativo 5V
E/S Digitales 20 7 PWM
Entradas analógicas 20
Memoria Flash 32KB 4KB usados por el bootloader
SRAM 2.5KB
EEPROM 1KB
Velocidad del Reloj 16MHz

Arduino Micro // Geniuino Micro

Microcontroller ATmega32U4
Voltaje 5V
Voltaje de entrada (recommendado) 7-12V
Voltaje de entrada (limite) 6-20V
Digital I/O 20
PWM 7
Canales analógicos 12
DC por I/O Pin 20 mA
DC por 3.3V Pin 50 mA
Flash Memory 32 KB (ATmega32U4) de los cuales 4 KB usedos por el bootloader
SRAM 2.5 KB (ATmega32U4)
EEPROM 1 KB (ATmega32U4)
Clock Speed 16 MHz

¿Con que se programa?

IDE Arduino

El IDE dea arduino se puede descargar desde la web de arduino

Aunque se puede programar con otros dependiendo del lenguaje, Netbeans, Matlab, Visual Studio.

Lenguaje en programarlo

Se programa mediante el uso de un lenguaje propio basado en el lenguaje de programación de alto nivel Processing que es similar a C++.

Interfaz con otro Software

Se puede comunicar un código hecho en arduino con codigo hecho en otros lenguajes

  • PHP
  • Python
  • Java
  • Gambas
  • C#
  • C++ (mediante libSerial o en Windows)
  • Ruby

Sintaxis básica

  • Delimitadores:;, {}
  • Comentarios: //, /* */
  • Cabeceras: #define, #include
  • Operadores aritméticos: +, -, *, /, %
  • Asignación: =
  • Operadores de comparación: ==, !=, <, >, <=, >=
  • Operadores Booleanos: &&, ||, !
  • Operadores de acceso a punteros: *, &
  • Operadores de bits: &, |, ^, ~, <<, >>
  • Operadores compuestos:
    • Incremento y decremento de variables: ++, --
    • Asignación y operación: +=, -=, *=, /=, &=, |=

Estructuras de control

  • Condicionales: if, if...else, switch case
  • Bucles: for, while, do. while
  • Bifurcaciones y saltos: break, continue, return, goto

Tipos de datos

  • void, boolean, char, unsigned char, byte, int, unsigned int, word, long, unsigned long, float, double, string, array.

Constantes

  • HIGH/LOW: representan los niveles alto y bajo de las señales de entrada y salida. Los niveles altos son aquellos de 3 voltios o más.
  • INPUT/OUTPUT: entrada o salida.
  • false (falso): Señal que representa al cero lógico. A diferencia de las señales HIGH/LOW, su nombre se escribe en letra minúscula.
  • true (verdadero): Señal cuya definición es más amplia que la de false. Cualquier número entero diferente de cero es "verdadero", según el álgebra de Boole, como en el caso de -200, -1 o 1. Si es cero, es "falso".
